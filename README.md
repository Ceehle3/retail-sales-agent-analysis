Retail Sales AI Agent Project

Project Objective
This project uses an AI agent to analyze retail sales data (Jan 2023 – Jan 2024) focusing on electronics, clothing, and beauty products to support business decision-making.

Tools Used
- SQL (Databricks / MySQL)
- AI Agent on databricks (Genie spaces)

Files Included
- report.docx → Full project explanation and findings
- dataset.csv → Retail sales dataset 
- README.md → Project overview

Agent Functionality
The AI agent answers business questions such as:
- Sales trends over time
- Best-performing product categories
- Monthly and seasonal comparisons

 Sample Questions
- Which product category performed best?
- What were peak sales months?
- How do categories compare over time?

Key Insights
- Electronics generated highest revenue
- Seasonal trends influenced sales peaks
- Clothing showed stable performance

Conclusion
The AI agent helps convert raw retail data into actionable insights for better decision-making.
